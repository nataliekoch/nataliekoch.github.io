Work

PawFinder

PawFinder is the first full-stack website I created for an assigment at Prime Academy. The goal of this project was to create a functional website on my own in two weeks.

I got my inspiration for this project when I was looking into adopting a pet of my own. I felt that the process could be greatly improved. I applied a masonry styled interface so you can scroll through more pets quickly. I added bright indicators for pets with special needs and compatability issues to optimaize users browsing time. I also made the site as simple as possible, so not registeration is needed to use and navigation is easy to use.

Equipment Manager App

The Equipment Manage App was my group project for Prime Academy. Along with 4 other students, we created an application for Sport Ngin that allows equipment managers an easy way to upload equipment and to reserve equipment out for sport events.

My role in this project was focused on front-end design. Using Sport Ngin's style guide, I designed and coded all HTML5/CSS3 for the application while making input mobile friendly on all devices.
